1 # HDIM
2 
3 HDIM is a toolkit for working with high-dimensional data that emphasizes
4 speed and statistical guarantees. Specifically, it provides tools for working
5 with the LASSO objective function.
6 
7 HDIM provides iterative solvers for the LASSO objective function
8  including ISTA, FISTA and Coordinate Descent. HDIM also provides FOS,
9  the Fast and Optimal Selection algorithm, a novel new method
10 for variable selection.
11 
12 HDIM is a product of the research conducted by
13 [Lederer and Hauser HDIM Group]( https://lederer.stat.washington.edu/ ).
14 
15 ## Supported Languages

30 # Installation
31 
32 ## C++ ( All Platforms )
33 
34 HDIM's native code base is cross-platform and header-only.
35  There is no installation step -- just clone the git repository and `#include`
36  the appropriate source files in your C++ projects. Just be sure to link against the
37 appropriate dependencies, as outlined in the *Dependencies* section below.
38 
39 ##### Dependencies
40 
41 HDIM's native code base depends on the following libraries.
42 
43 * [Eigen3](http://eigen.tuxfamily.org/index.php?title=Main_Page)
44 
45 These libraries will need to be installed in order to use the native C++
46  or any of the wrappers.
47 
48 ## Wrappers
49 
50 Using the Python or R wrapper requires additional dependencies and build steps
51 compared to using the native code base. Currently both wrappers require building from source --
52  in the future we hope to provide users with more convenient installation options.
53 
54 #### R ( Linux & Windows )
55 
56 Visit the [HDIM-R](https://github.com/LedererLab/HDIM-R) repository for
57 installation instructions for the `HDIM` R wrapper.
58 
59 ##### Python ( Linux & OS X)
60 
61 Visit the [HDIM-Py](https://github.com/LedererLab/HDIM-Py) repository for
62 installation instructions for the `HDIM` Python wrapper.
